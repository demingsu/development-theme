## zznode-cf V0.0.1
###### The zznode-cf exports as a customize module for form

## Installation
###### Using npm:
```
$ npm i -D zznode-cf
```

###### In vue main module:
```
import ZznodeCF from "zznode-cf";
import "zznode-cf/lib/zznode-cf.css";
Vue.use(ZznodeCF);
```

## Support
Tested in Chrome45, IE10+
